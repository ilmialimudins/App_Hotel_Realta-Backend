# END POINT PURCHASING

## STOCK

Get All Data :

```bash
GET : localhost:{PORT}/stock
```

Get One Data :

```bash
GET : localhost:{PORT}/stock/{stockName}
```

Create Data :

```bash
POST : localhost:{PORT}/stock
```

Update Data :

```bash
PUT : localhost:{PORT}/stock/{stockName}
```

Delete Data :

```bash
# DELETE: localhost:{PORT}/stock/{stockName}
```
